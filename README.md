# IA Conversacional Simples

Este é um projeto de IA conversacional em Python utilizando TensorFlow e Keras para classificar intenções e gerar respostas com modelos neurais.

---

## 🐍 Requisitos

- Python 3.10 (recomendado)
- TensorFlow 2.19.0
- NumPy 2.1.3
